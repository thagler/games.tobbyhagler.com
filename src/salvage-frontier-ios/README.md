# Salvage Frontier iOS Bootstrap

This directory contains the initial iOS bootstrap source scaffold for Task 0001.

## Expected project path

- `SalvageFrontier.xcodeproj` (to be created/opened in Xcode)
- App source root: `SalvageFrontier/`

## Included in bootstrap

- SwiftUI app entry point
- SpriteKit-backed transit test view
- minimal movement + auto-fire test scene
- lightweight configuration constants

## Note

The current CLI environment does not have full Xcode installed, so project generation/build validation must be completed on a machine with Xcode.
