# 0001 Bootstrap Project

## Objective
Set up the project scaffold and a minimal playable test scene.

## Constraints
- prioritize speed to first playable
- no advanced architecture work

## Acceptance Criteria
- project runs locally
- at least one playable placeholder scene exists
- movement + shooting test space is available

## Non-Goals
- final game architecture
- production art

## Test Steps
1. Run the project locally.
2. Enter the placeholder scene.
3. Verify basic movement and shooting input.
