# Core Loop

Transit combat
→ Land on outpost
→ Extract resources
→ Build/defend
→ Refuel/rearm
→ Launch
→ Choose next route

## Fun Criteria
- Transit is immediately legible and rewarding.
- Landing creates a clear shift in priorities.
- Outpost decisions have tradeoffs (defense vs extraction vs timing).
- Launch timing feels tense but fair.
- Route choice sets up meaningful next-run differences.

## Codex Build Rules
1. Prioritize playable loop over architecture purity.
2. Prefer one complete vertical slice over broad unfinished systems.
3. Do not add mechanics without explicit acceptance criteria.
4. Keep data-driven structure where easy, avoid premature complexity.
5. After each milestone, summarize built work, remaining work, and risks.
