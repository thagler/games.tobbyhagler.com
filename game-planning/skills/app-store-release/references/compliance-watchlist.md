# Compliance Watchlist

## Time-Sensitive Policy Checks

Run these checks for each release cycle:

- Apple upcoming requirements (Xcode/SDK minimum upload requirements).
- App Review policy sections affecting monetization and account flows.
- Third-party SDK privacy-manifest/signature requirements.

## iOS Monetization Rules

- Digital goods and unlocks must follow Apple in-app purchase policy where required.
- Match app behavior and metadata disclosures to implemented purchase flows.

## Operational Practices

- Keep a release log with review feedback and remediation notes.
- Separate urgent hotfix release workflows from regular feature releases.
- Confirm legal/compliance owner sign-off for policy-sensitive features.
